# Flask Starter Base

Este repositório é uma **base simples para projetos utilizando Flask**.  
Ele existe para servir como ponto de partida para novas aplicações, evitando a necessidade de recriar configurações básicas sempre que um novo projeto começa.

A ideia é fornecer uma estrutura mínima, clara e reutilizável para desenvolver APIs ou aplicações web utilizando Flask.

---

# Objetivo do Projeto

Este projeto foi criado para:

- Servir como **template inicial para aplicações Flask**
- Demonstrar **como organizar rotas**
- Mostrar **como integrar e manipular um banco de dados**
- Facilitar a criação rápida de novos projetos

Em vez de começar do zero, basta clonar o repositório e adaptar conforme necessário.
